# Security Policy

## Supported Versions

This project currently supports the latest version on the default branch.

## Reporting A Vulnerability

Please do not open public issues for security-sensitive problems.

Instead:

- describe the issue clearly
- include impact and reproduction details if possible
- share the affected version or commit

If you are reporting through GitHub, prefer private security reporting when available for the repository.

We will review reports as quickly as possible and coordinate a fix before public disclosure when appropriate.
